# Papyrus Notes

**Proje Açıklaması:**  
Papyrus Notes, kendi notlarınızı ekleyip, düzenleyip ve silebileceğiniz basit bir React uygulamasıdır.  
React + Tailwind CSS ile geliştirilmiş olup, dark/light modu ve localStorage desteği içerir.

## Kurulum & Çalıştırma

1. Projeyi klonlayın:
```bash
git clone https://github.com/eceboylu/my-notes-app.git
cd my-notes-app
Bağımlılıkları yükleyin:

npm install
Uygulamayı başlatın:

npm start
Tarayıcıda http://localhost:3000 adresini açın.

Production build oluşturmak için:

npm run build
